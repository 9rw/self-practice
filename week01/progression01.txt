Week1 - Web Stopwatch

รายละเอียด
- สร้างเว็บจับเวลา (Stopwatch)
- ผู้ใช้สามารถเริ่ม หยุด และรีเซ็ตเวลาได้
- แสดงเวลาที่ผ่านไปแบบเรียลไทม์

Reflect
- ได้ฝึกใช้งานฟังก์ชัน setInterval และการจัดการ state
- ได้เรียนรู้การแปลงเวลาในรูปแบบมิลลิวินาทีให้เป็นชั่วโมง นาที วินาที
- เข้าใจการเข้าถึงและจัดการองค์ประกอบของหน้า HTML ผ่าน JavaScript มากยิ่งขึ้น
