Week 3 - Object Destructuring and CRUD Operations

รายละเอียด:
- สร้างฟังก์ชันสำหรับการจัดการผู้ใช้ (CRUD: Create, Read, Update, Delete)
- ใช้ Object Destructuring เพื่ออัปเดตค่าภายใน Object เมื่อรับพารามิเตอร์จากผู้ใช้
- เพิ่มฟังก์ชัน Validation(การตรวจสอบข้อมูล) และจัดการข้อผิดพลาดที่เกิดขึ้น

Reflect:
- เข้าใจเรื่อง Object Destructuring การดึงค่าจาก Object โดยตรงเพื่ออัปเดตข้อมูลของผู้ใช้ มากยิ่งขึ้น
  เช่น ในฟังก์ชัน editUser และ addUser
- ฝึก Validation(การตรวจสอบข้อมูล) เรียนรู้การตรวจสอบข้อมูลที่ได้รับจากผู้ใช้ก่อนที่จะทำการอัปเดตหรือเพิ่มข้อมูล
  เช่น การตรวจสอบอีเมล (ผ่านฟังก์ชัน checkEmail) และการตรวจสอบว่า status อยู่ในช่วงที่ถูกต้อง
- ในกรณีที่ข้อมูลไม่ถูกต้อง จะใช้ error logs เพื่อแจ้งข้อผิดพลาดที่เกิดขึ้น
- การใช้ Array method ต่างๆ
  เช่น find(), findIndex(), filter(), และ splice() เพื่อค้นหาข้อมูล, ตรวจสอบ, แก้ไข, และลบข้อมูลของผู้ใช้
- ได้เรียนรู้การจัดการข้อผิดพลาดเมื่อข้อมูลไม่ถูกต้องโดยการใช้ Object 
  ที่เก็บข้อมูลของผู้ใช้พร้อมกับ success flag และ error logs เพื่อให้ฟังก์ชันแต่ละตัวตอบกลับได้ชัดเจน
